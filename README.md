# 🌱 Nursery

A beautiful static website for plant care and gardening services, hosted on GitHub Pages.

## 🌐 Live Site

Visit our website: [https://infiniti-dev.github.io/nursery-/](https://infiniti-dev.github.io/nursery-/)

## ✨ Features

- **Responsive Design**: Optimized for all devices and screen sizes
- **Modern UI**: Clean, professional design with smooth animations
- **SEO Optimized**: Meta tags and semantic HTML structure
- **Fast Loading**: Optimized static assets and modern CSS
- **Interactive**: Smooth scrolling navigation and hover effects

## 🚀 Deployment

This site is automatically deployed to GitHub Pages using GitHub Actions whenever changes are pushed to the main branch.

### Manual Deployment

The site uses GitHub Pages' built-in static site hosting. The deployment workflow is configured in `.github/workflows/static.yml`.

## 📁 Structure

```
├── index.html          # Main homepage
├── styles.css          # CSS styling
├── script.js           # JavaScript functionality
├── .github/
│   └── workflows/
│       └── static.yml  # GitHub Pages deployment workflow
└── README.md           # This file
```

## 🛠️ Development

To work on this project locally:

1. Clone the repository
2. Open `index.html` in your browser
3. Make changes to the HTML, CSS, or JavaScript files
4. Push changes to trigger automatic deployment

## 📱 Sections

- **Home**: Welcome hero section with call-to-action buttons
- **About**: Information about nursery services and features
- **Services**: Overview of available plant care services
- **Contact**: Contact information and business hours

The site uses a green color palette reflecting nature and plants:
